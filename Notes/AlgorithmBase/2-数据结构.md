###  1.1 单链表

​	最常用的单链表：邻接表（存储图和树）

​	数组模拟单链表：用e[N]表示值，ne[N]表示指针，以-1表示空集

​	单链表的操作：初始化、插入、删除 

```java
public class Main{
    // 初始化
    public static void init(){
        head = -1;
        idx = 0;
    }
	// 在头节点之前添加节点
    public static void add_to_head(int x){
        e[idx] = x;
        ne[idx] = head;
        head = idx ++;
    }
	// 在第k个节点后添加节点
    public static void add(int k, int x){
        e[idx] = x;
        ne[idx] = ne[k];
        ne[k] = idx ++;
    }
	// 删除第k个节点之后的节点
    public static void delete(int k){
        ne[k] = ne[ne[k]];
    }
}
```
