## 4 树和图的遍历

### 4.1 DFS

```java
	public static int dfs(int u){
        st[u] = true;
        
        int size = 0, sum = 0;
        for(int i = head[u]; i != -1; i = ne[i]){
            int j = e[i];

            if(!st[j]){
                int s = dfs(j);
                size = Math.max(size, s);
                sum += s;
            }
        }

        size = Math.max(size, n - sum - 1);
        res = Math.min(size, res);

        return sum + 1;
    }
```

### 4.2 BFS

```java
	public static int dfs() {
        int res = -1;
        Queue<Integer> queue = new LinkedList<>();

        queue.add(1);
        d[1] = 0;

        while (!queue.isEmpty() & res == -1) {
            int t = queue.peek();
            queue.remove();

            for (int i = head[t]; i != -1; i = ne[i]) {
                int j = e[i];

                if (d[j] == -1) {
                    queue.add(j);
                    d[j] = d[t] + 1;
                }
            }

            if (t == n)
                res = d[n];
        }
        
        return res;
    }
```

## 5 有向图的拓扑排序

* 拓扑图：有向无环图

```java
	public static boolean topSort(int n) {
        int hh = 0, tt = 0;
        // 先将入度为0的点插入队列
        for (int i = 1; i <= n; i++) {
            if (in[i] == 0) 
                q[tt++] = i;
        }
        
        while (hh < tt) {
            // 获取队头
            int t = q[hh++];

            for (int i = head[t]; i != -1; i = ne[i]) {
                int j = e[i];
                // 弹出队头后更新入度
                in[j]--;    
                // 将入度为0的点插入队列
                if (in[j] == 0)
                    q[tt++] = j;
            }
        }
        
        return tt == n;
    }
```
